
## 功能说明

### 1. 商品展示
- 展示6个商品，分为3个类别：电子产品、服装配饰、图书文具
- 每个商品显示图片、名称、描述、价格和加入购物车按钮

### 2. 商品筛选
- 通过分类按钮筛选商品
- 支持筛选全部商品或按类别筛选

### 3. 搜索功能
- 在搜索框中输入关键词搜索商品
- 支持按商品名称或描述搜索
- 搜索结果会高亮显示

### 4. 购物车功能
- 添加商品到购物车
- 查看购物车中的商品
- 调整商品数量
- 从购物车中移除商品
- 购物车数据保存在本地存储中，页面刷新后不会丢失

### 5. 通知系统
- 添加商品到购物车时显示成功通知
- 搜索结果显示搜索结果通知
- 结算时显示结算结果通知

### 6. 响应式设计
- 适配不同屏幕尺寸的设备
- 在手机和平板上显示效果良好

## 如何运行项目

1. 将所有文件下载到本地
2. 确保文件结构保持不变
3. 使用浏览器打开index.html文件
4. 确保浏览器支持ES6模块（现代浏览器都支持）

## 注意事项

1. 由于使用了ES6模块，需要在服务器环境下运行（或使用支持本地文件访问的浏览器设置）
2. 商品图片来自Unsplash，需要网络连接才能显示
3. 字体图标来自Font Awesome CDN，需要网络连接
